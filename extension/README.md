# 七牛内容助手 - 浏览器扩展

## 安装步骤

1. 打开 Chrome 浏览器，地址栏输入：`chrome://extensions/`

2. 右上角开启 **「开发者模式」**

3. 点击 **「加载已解压的扩展程序」**

4. 选择 `七牛2/extension` 文件夹

5. 扩展安装成功，工具栏会出现一个 **"七"** 图标

## 使用方法

1. 在七牛App中创建内容并选择要发布的平台
2. 点击 **「一键发布」**
3. 各平台编辑器会在浏览器中打开
4. 在每个平台编辑器页面上，点击工具栏的 **「七」** 图标
5. 点击弹出窗口中的 **「一键填入」** 按钮
6. 内容自动填入！

## 支持的平台

- B站专栏 (member.bilibili.com/read)
- B站视频投稿 (member.bilibili.com/upload)
- B站动态 (bilibili.com/v/publish)
- 微信公众号 (mp.weixin.qq.com)
- 知乎专栏 (zhuanlan.zhihu.com)
- 小红书 (creator.xiaohongshu.com)

## 注意事项

- 首次使用需要先在各平台登录
- 视频和图片需要手动上传
- 如果填入失败，请刷新页面后重试
